# Soilboy Clone a Frontend Learning Project.

A visual frontend recreation of the beautifully designed [Soilboy](https://www.soilboy.sg/) e-commerce platform. 

This project was built strictly as a **learning exercise** to practice responsive layout design, CSS styling patterns, and component architecture using vanilla web technologies.

## ⚠️ Important Disclaimer
This is a **non-commercial, portfolio-only project**. 
* **No Backend / Authentication:** There is no functional sign-up, user login, database persistence, or checkout capability.
* **Asset Notice:** All photography, branding, and product copy belong entirely to Soilboy. This repository exists solely to showcase frontend layout and design replication skills.

## 🛠️ Built With
To challenge myself and master core web fundamentals, this project relies entirely on raw web standards without heavy frameworks (like React or Vue):

* **HTML5:** markup for structure and accessibility.
* **CSS3:** Custom properties, Flexbox, and CSS Grid for a fluid, responsive layout that mimics the original site's feel.
* **Vanilla JavaScript:** Simple DOM manipulation for interactive UI elements.

## 🚀 Key Features Practiced
* **Responsive Design:** Ensuring the site functions on everything from mobile screens and desktop monitors.
* **Interactive Components:** Creating user interface interactions using raw JavaScript.

## 📦 How to Run Locally
This is a static frontend site, it requires no installation or build steps simply clone the project locally to run.
